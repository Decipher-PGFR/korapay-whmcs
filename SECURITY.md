# Security Policy

## Reporting a Vulnerability

Please report suspected vulnerabilities privately by email:

security@decipher.ng

Do not open a public GitHub issue for a security vulnerability. Include the affected module, a clear reproduction path, expected impact, and any relevant request or webhook samples with secrets removed.

We aim to acknowledge reports within 72 hours and will coordinate remediation and disclosure timing with the reporter where appropriate.

## Scope

This policy covers the Korapay WHMCS gateway module files in this repository.

## Sensitive Data

Never include live API keys, webhook secrets, customer data, card data, WHMCS admin paths, server IPs, SSH usernames, or internal deployment details in issues, pull requests, logs, or screenshots.
